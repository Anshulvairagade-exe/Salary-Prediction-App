Salary Prediction App

Live Demo:
(https://salary-prediction-app-a8ba9eeaxx9uzikudmt9li.streamlit.app/)
Description

A Machine Learning web app that predicts an employee’s salary based on age, gender, education level, job title, and years of experience.

Features:
Salary prediction using Random Forest
Handles missing values
Encodes categorical data
Simple user interface

Tech Stack:
Python
Pandas
NumPy
Scikit-learn
Streamlit

Project Files:
app.py
random_forest_regressor_model.pkl
requirements.txt

Run the App
pip install -r requirements.txt
streamlit run app.py
